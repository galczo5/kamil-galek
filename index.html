NAME top

kamil-galek.pl - display information about developer skills, work experience and freelance projects

SYNOPSIS top

kamil-galek.pl [OPTIONS...] [COMMAND]

DESCRIPTION top

Hi! Welcome on my site. Here you can get informations about me, my work and my interests. If you don't like to type commands please scroll to NOTES section and download all informations as pdf file. Good luck and have fun :)

OPTIONS top

Here is a table of the generic command line switches accepted by the kamil-galek.pl command:

--help, -h display this help and exit

--all, -a display all informations

--pdf, -p export to pdf file

--asc sort items in ascending order

--desc sort items in descending order

--open-external-content open external content such as images or hyperlinks

COMMANDS top

personal-informations print my personal information

skills print my skills

experience print my experience information

freelance print my freelance projects

SEE ALSO top

github-profile my github profile

blog my blog website

reddit-profile my reddit profile
